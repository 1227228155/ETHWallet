## Features
* 支持多钱包
* 发送/请求付款
* 支持代币
* 消息通知
* 交易历史
* 钱包导入/导出
* 不需要注册登录
* 支持ERC20
* 支持英语
* 支持中文
* 密码保护
* 密码重置

## 后续计划
* 指纹保护
* 支持只读钱包
* 支持助记词


根据[Trust](https://github.com/TrustWallet/trust-wallet-android-source)修改优化
